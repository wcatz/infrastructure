# Ansible Infrastructure Management

This repository contains the Ansible framework for managing infrastructure.

## Structure

- **inventories/**: Contains inventory files for different environments.
  - **development/**: Inventory for development/staging.
  - **production/**: Inventory for production.
- **roles/**: Contains reusable Ansible roles for various tasks.
- **playbooks/**: High-level playbooks for orchestrating tasks.

## Getting Started

1. Clone the repository.
2. Navigate to the ansible-infra directory.
3. Modify the inventory files as needed.
4. Run the playbooks to manage your infrastructure.
